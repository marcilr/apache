Compile and Install Apache

Apache Software Foundation

  1. Download latest stable Apache web server source.
    1. Open Apache download site in web browser:
      https://httpd.apache.org/download.cgi
    2. Get path for latest stable Apache web server source:
      https://dlcdn.apache.org/httpd/httpd-2.4.53.tar.bz2
    3. Create local directory to download apache source to:
      $ mkdir -p /usr/local/src/00TARBALLS/net/apache
    4. Change local directory and download the lastest apache source code:
      cd /usr/local/src/00TARBALLS/net/apache
      wget https://dlcdn.apache.org/httpd/httpd-2.4.53.tar.bz2
  2. Change to src/ directory and expand the apache tarball.
    $ cd /usr/local/src $ tar jxvpf 00TARBALLS/net/apache/httpd-2.4.53.tar.bz2 ar.bz2
    httpd-2.4.53/
    httpd-2.4.53/acinclude.m4
    httpd-2.4.53/README.cmake
    httpd-2.4.53/INSTALL
    ...
    httpd-2.4.53/test/pyhttpd/htdocs/test1/007/007.py

    $ du -h httpd-2.4.53/
    ...
    49M httpd-2.4.53/
    $ cd httpd-2.4.53
    $ find . | wc -l 3253

    Expanded apache source looks good.

  3. Compile apache source code following httpd-2.4.53/INSTALL/ instructions.
    $ cd /usr/local/src/httpd-2.4.53
    $ less INSTALL

    APACHE INSTALLATION OVERVIEW
    ...
    $ ./configure --prefix=PREFIX
    $ make
    $ make install
    $ PREFIX/bin/apachectl start
